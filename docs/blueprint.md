# **App Name**: WATTch

## Core Features:

- Realtime Data Display: Display electricity data in realtime for each load type (light, medium, heavy, universal).
- Load Graph: Present each load's data as a dynamic graph, updating in real-time.
- Active Status Indicator: Display an 'Active' status for each load if receiving data, otherwise, after 3 seconds of no new data, show 'Inactive'.
- Power Control: Include a power button for each load to control and monitor its status.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to represent energy and technology.
- Background color: Light gray (#E0E0E0) to provide a neutral and clean interface.
- Accent color: Bright purple (#BE52F2) to highlight active elements and buttons.
- Font: 'Inter', a sans-serif font providing a modern and neutral look suitable for both headlines and body text.
- Use minimalist, vector-based icons to represent each load type and interactive elements.
- Dashboard layout with clear separation of each load's data and controls for easy monitoring.
- Subtle animations for data updates on the graphs and status changes for a dynamic feel.
